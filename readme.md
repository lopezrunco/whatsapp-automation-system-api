# API REST with framework Express (MongoDB)

## Instructions

Crete .env file with environment variables

```bash
PORT=3000
JWT_KEY=super_secret_key
DB_HOST=localhost
DB_PORT=3306
DB_NAME=wappautomate
DB_USER=root
DB_PASSWORD=root
```

```bash
# Install dependencies
npm i

# Insert demo data
npm run seed

# Start API
npm run dev
```
